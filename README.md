# OpioidOracle

OpioidOracle is a predictive analytics framework designed to forecast Long-Acting (LA) opioid prescribing rates using historical Medicaid prescription data. Built with Facebook Prophet, the project performs time-series analysis on opioid prescribing trends across multiple geographic regions and healthcare plan types.

# Features
Time-series forecasting with Facebook Prophet
Geographic-level opioid trend analysis
Lag and rolling-window feature engineering
Forecast visualization and trend interpretation
Model evaluation using MAE, MSE, RMSE, MAPE, and R²
Regional forecasting for Medicaid opioid prescription data

Objective

To identify long-term opioid prescribing patterns and generate reliable forecasts that can assist healthcare researchers, policymakers, and public health organizations in understanding future opioid utilization trends.
